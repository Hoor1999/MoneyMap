# MoneyMap

MoneyMap is a simple console application built with C# to help users track and manage their personal expenses efficiently. 

## Features
* **Add & Delete Expenses:** Easily record new transactions or remove outdated/wrong entries.
* **Category Filtering:** Filter and view expenses based on their type, such as Shopping, Transportation, Food, etc.
* **Basic Insights:** Helps you keep track of where your money is going.

## Technologies Used
* **Language:** C#
* **Framework:** .NET

## How to Run
1. Clone the repository.
2. Open the solution file `MoneyMap.sln` in Visual Studio or your favorite IDE.
3. Build and run the project.
